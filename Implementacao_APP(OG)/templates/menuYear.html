Dashboards e Inteligência Artificial na Proteção de Grupos Vulneráveis: Análise de Dados do Disque 100

Este trabalho aborda os resultados do desenvolvimento de uma plataforma computacional para análise exploratória de dados provenientes do Disque 100, disponibilizados pela Ouvidoria Nacional dos Direitos Humanos. A proposta evidencia o potencial da visualização de dados como ferramenta capaz de fornecer insights relevantes para a formulação de políticas públicas. Adicionalmente, a ferramenta é integrada à API Gemini, um chatbot desenvolvido pelo Google, com o objetivo de gerar análises automatizadas e sugerir estratégias de prevenção e intervenção com base nos dados analisados. Dessa forma, o trabalho ressalta o papel das tecnologias de visualização de dados e das aplicações web como aliadas na análise de grandes volumes de informações sociais. O uso combinado de aplicações web, análise de dados e IA desponta como uma estratégia promissora para fortalecer a resposta institucional diante de problemas sociais complexos.
